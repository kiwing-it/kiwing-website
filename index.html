Inicio

¡Le damos la bienvenida a Kiwing! Aquí encontrará un equipo de excelentes desarrolladores, pero por sobretodo grandes personas que sienten una profunda pasión por lo que hacen.

- Nuestra empresa

Somos una empresa de desarrollo de Software a la medida con sede en Córdoba, Argentina. Trabajamos con clientes de todo el mundo, ofreciendo soluciones informáticas únicas para quienes desean destacar su negocio entre la multitud. La atención personalizada es uno de nuestros pilares principales.

- Nuestro compromiso es la calidad

- Nuestras habilidades

Mauris commodo

Equipo multidisciplinario con Core en PHP. Hacemos trabajos con híbridos multiplataforma como Ionic 3, ya que consideramos que es una tecnología que se está ganando el mercado mobile. Tenemos experiencia en el desarrollo de: API REST con estándar Api Json Specification, GIT, MySQL, Arquitectura Hexagonal, PHP Frameworks (Laravel, Symfony), PHP Micro-Frameworks (Lumen, Slim), PHP-FIGs, y Frontends desacoplados con tecnologías como Vue, conexión con Cloud Communications Platform como Twilio para realizar videollamadas, entre otros. Otro aspecto destacable es que la mayoría de los devs son aficionados a Inteligencia Artificial gracias a la orientación ingenieril con la que cuentan. ¡Nuestros productos son de calidad y están hechos con los mejores estándares!
